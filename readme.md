## 📖 Sobre o Projeto

O **DevLinks** é uma aplicação desenvolvida com foco em prática de desenvolvimento front-end, utilizando HTML, CSS e JavaScript.

O projeto foi criado durante meus estudos na plataforma Rocketseat com o objetivo de aprimorar conhecimentos em estruturação de páginas web, estilização moderna e interatividade com JavaScript.

A aplicação apresenta uma interface moderna, responsiva e organizada para centralizar links e informações de forma prática e visualmente agradável.
